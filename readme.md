# Robot for oracle and tokenManager

## Prepare

node v12.18.3

## Install

```
npm i
```

## Add or update Token

Change config in config/*.json

for testnet

```
npm run tokenPair_testnet
```

for mainnet

```
npm run tokenPair_production
```

## Start robot

for testnet

```
npm run robot_testnet
```

for mainnet

```
npm run robot_production
```
